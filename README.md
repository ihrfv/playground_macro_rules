This is a pet project to play around creating my own `macro_rules!`.
It is a continuation of the [video tutorial](https://www.youtube.com/watch?v=q6paRBbLgNw&list=PLqbS7AVVErFiWDOAVrPt7aYmnuuOLYvOa&index=2) by Jon Gjengset where he shows/explaines how to write/is written `vec![]` macro.
This a build up on top of his "Crust of Rust" where I attempt to build similar macro for `HashSet`.

Other references:
* [The Little Book of Rust Macros](https://lukaswirth.dev/tlborm/)
